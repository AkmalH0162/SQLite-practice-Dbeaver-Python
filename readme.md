# Chinook SQL Practice

This project contains SQL practice work using the Chinook sample SQLite database. It includes queries and examples focused on data cleaning, transformation, reporting, and basic data manipulation.

## Project Contents

- `practice.ipynb`:
  Jupyter notebook used to connect to the SQLite database, run SQL files, and load outputs into pandas DataFrames.

- `query1.sql`:
  Basic SQL query practice.

- `01_data_cleaning.sql`:
  Examples of cleaning text fields, handling NULL values, and standardizing data.

- `02_data_transformation.sql`:
  Examples of transforming invoice data into analysis-friendly columns.

- `03_create_summary_table.sql`:
  Creates a customer spending summary table from invoice data.

- `04_update_example.sql`:
  Demonstrates safe updates using a copied table.

- `05_sales_report.sql`:
  Builds a sales report by track, album, artist, and genre.

- `06_create_view.sql`:
  Creates a reusable SQL view for customer invoice analysis.

## Tools Used

- SQLite
- DBeaver
- Python
- pandas
- Jupyter Notebook

## Purpose

The purpose of this project is to practice SQL querying and understand how SQL can be used for:

- data cleaning
- data transformation
- aggregation
- reporting
- creating views and summary tables

## Notes

The database used is the Chinook sample database. Some queries are designed for learning and demonstration purposes.
